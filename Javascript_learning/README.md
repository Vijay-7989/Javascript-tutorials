# JavaScript Learning & Teaching

A comprehensive beginner-friendly JavaScript tutorial repository designed for learning and teaching fundamental JavaScript concepts.

## 📁 Project Structure

### Files Overview

#### 1. **hello.js**
A simple introductory program that prints "Hello, World!" to the console.
- Demonstrates basic `console.log()` usage
- First step in JavaScript programming

#### 2. **simpleSum.js**
A basic arithmetic program that adds two numbers and displays the result.
- Shows variable assignment
- Demonstrates basic arithmetic operations
- Introduces string concatenation with `console.log()`

#### 3. **variables.js**
Comprehensive guide to JavaScript variable declaration keywords.
- **`var`** - Function-scoped, can be redeclared
- **`let`** - Block-scoped, cannot be redeclared
- **`const`** - Block-scoped, cannot be reassigned
- Includes practical examples and common patterns

## 🚀 Getting Started

### Prerequisites
- Node.js installed on your system

### Running the Scripts

```bash
# Run hello.js
node hello.js

# Run simpleSum.js
node simpleSum.js

# Run variables.js
node variables.js
```

- **Data Types** - Working with strings, numbers, and null values
- **String Concatenation** - Combining strings and variables
- **Arithmetic Operations** - Basic mathematical calculations

## 📝 Key Concepts & Best Practices

### Variable Declaration Keywords
- **`var`** - Function-scoped, can be redeclared (legacy, avoid in modern code)
- **`let`** - Block-scoped, cannot be redeclared (modern choice)
- **`const`** - Block-scoped, cannot be reassigned (use by default)
